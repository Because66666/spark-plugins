## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/xLevitate/spark-plugins.git
cd spark-plugins
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
