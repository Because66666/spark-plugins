# SparkPlugins

一个现代高效的Minecraft插件下载器，可以从SpigotMC和Modrinth等多个来源搜索和下载插件。Because66666进行了全局汉化。不用于PR.

## 功能

- 🔍 从多个来源搜索插件
- ⚡ 快速响应的界面
- 🌙 简洁的深色主题设计
- 📦 批量插件下载
- 🔄 版本选择支持
- 💻 现代网页技术

## 技术栈

- **框架**: Next.js
- **样式**: Tailwind CSS
- **状态管理**: React Context
- **API**: Spiget, Modrinth

## 开始使用

1. 克隆仓库:
```bash
git clone https://github.com/Because66666/spark-plugins.git
cd spark-plugins
```

2. 安装依赖:
```bash
npm install
# 或
yarn install
```

3. 运行开发服务器:
```bash
npm run dev
# 或
yarn dev
```

4. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 开发

项目结构遵循Next.js约定:

```
src/
├── app/             # 应用路由页面
├── components/      # React组件
├── contexts/        # React上下文
└── types/          # TypeScript类型
```
